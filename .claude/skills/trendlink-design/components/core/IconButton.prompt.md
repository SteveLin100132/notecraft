Icon-only button for search, close, pagination, and toolbar actions. Always pass `label` for accessibility.

```jsx
<IconButton variant="accent" label="搜尋"><SearchIcon /></IconButton>
<IconButton variant="soft" shape="rounded" label="下一步"><ChevronRight /></IconButton>
```

Variants: `solid` (navy), `accent` (gold), `soft` (pale blue), `ghost`, `outline`. The blue search circle in the site header is `accent`/`solid` circle.
