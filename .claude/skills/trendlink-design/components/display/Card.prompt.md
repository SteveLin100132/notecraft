Primary content surface. Use `accent` for feature cards (matches the site's top-stripe service cards), `hoverable` for clickable grids. `CardIcon` is the rounded icon medallion.

```jsx
<Card accent="blue" hoverable>
  <CardIcon tone="blue"><SalaryIcon/></CardIcon>
  <h3>一鍵發薪 雲端人資系統</h3>
  <p>計薪、排班、休假、出勤一次搞定。</p>
</Card>
```
