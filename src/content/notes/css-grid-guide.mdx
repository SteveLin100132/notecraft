Flexbox 處理一維排列，Grid 則是真正的二維系統。當你需要同時控制行與列的對齊，Grid 幾乎總是更乾淨的選擇。

## 核心概念

- `grid-template-columns` / `rows` 定義軌道（track）。
- `fr` 單位按比例分配剩餘空間。
- `gap` 取代了過去用 margin 製造間距的繁瑣。
- `minmax()` 與 `auto-fit` / `auto-fill` 是響應式排版的關鍵。

```css
.cards {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(240px, 1fr));
  gap: 1.5rem;
}
```

搭配 minmax 與 auto-fit，這段就能讓卡片在容器變寬時自動增加欄數，不需任何媒體查詢。
